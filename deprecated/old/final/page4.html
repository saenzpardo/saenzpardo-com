The Future of the Web


Yet to be determined...

Although the future of the internet cannot be easily determined. I believe we have come full circle. Computers started with one centralized server and many workstations connected to that mainframe computer. This is what the Chrome Book has become. Progressive web applications and large data centers coupled with increased connection speeds, powerful processors, faster solid-state drives and cheaper ram with higher capacity have created a portable workstation that doesn’t need to have native applications installed. This is leading to the advancement of infrastructure as a service, software as a service, desktop as a service and many other online services. This allows for the rapid scalability if a business quickly grows and then quickly reduces size.


Gaming as a service will quickly take hold. You no longer need expensive hardware to play the latest video games. This is an industry that brought in over $150 billion dollars in 2019. They have the ability to make more money if they can reach users that can’t afford an expensive game system or powerful computer with the latest graphics cards. With cloud gaming services like GeForce Now, you can play the latest games on almost any device.


The increase in connectivity will also allow for real time data tracking that will increase machine learning opportunities. Autonomous cars would instantly be able to communicate to each other when traffic delays are present and re-calculate more efficient routes. I believe machine learning will soon take over many positions, including front-end developer jobs. We will rely more on machine learning technology for economic decisions, public transportation, managing a large corporate computer network and routing packages efficiently. It really can be applied to almost anything that is created or consumed. I just hope we use the data as a consideration and do not become reliant on the technology.
